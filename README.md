# Trabajo Fin de Grado: Modelo de Cadenas de Markov Ocultas de tiempo continuo para la detección de tumores de cáncer de colon
Este repositorio contiene el material desarrollado para el Trabajo de Fin de Grado "Modelo de Cadenas de Markov Ocultas de tiempo continuo para la detección de tumores de cáncer de colon", incluyendo la memoria completa del proyecto, el código fuente utilizado para el desarrollo y entrenamiento del modelo, así como los recursos necesarios para reproducir los experimentos y resultados presentados en el documento.

## Contenido del repositorio
- Memoria del TFG: documento final del trabajo en formato PDF.
- Código del modelo: archivo Rmarkdown con el código e instrucciones para la implementación completa del modelo utilizado en el proyecto.
- Datos: archivos con los conjuntos de datos utilizados en el trabajo, necesarios para reproducir los experimentos y resultados presentados.
- Configuraciones: archivos de configuración y parámetros requeridos para la correcta ejecución del modelo y la replicación de los análisis.

## Objetivo

El objetivo de este repositorio es facilitar la transparencia, reproducibilidad y difusión del trabajo realizado, permitiendo a otros estudiantes e investigadores comprender, replicar y extender los resultados obtenidos.

## Requisitos

- R >= 4.4.0
- RStudio (opcional)
- Paquetes de R especificados en `renv.lock`

## Entorno utilizado

- R 4.5.0
- NIMBLE 1.4.1
- Sistema operativo: Windows 11
